# Contract Vehicles

Contract vehicles are pre-negotiated procurement mechanisms (GWAC, IDIQ, BPA, etc.) that can be linked to opportunities.

## Common Vehicle Types

- **GSA MAS** (Multiple Award Schedule)
- **OASIS / OASIS+** (One Acquisition Solution for Integrated Services)
- **SeaPort-NxG** (Navy IDIQ)
- **CIO-SP3 / CIO-SP4** (IT services)
- **STARS III** (8(a) IT GWAC)
- **Alliant 2** (IT GWAC)

## Creating a Vehicle

1. Navigate to **Vehicles**.
2. Click **Add Vehicle**.
3. Fill in the fields:
   - **Name** -- Vehicle name.
   - **Agency** -- Issuing agency.
   - **Contract Number** -- Your contract number on this vehicle.
   - **Expiration Date** -- When the vehicle expires.
   - **Ceiling Value** -- Maximum value available.
   - **Prime or Sub** -- Whether you hold this vehicle as prime or subcontractor.
   - **Notes** -- Free-text notes.
4. Click **Save**.

## Linking to Opportunities

When creating or editing an opportunity, select applicable contract vehicles from the vehicle picker. This helps track which vehicles are being used for pursuit.

## Viewing Vehicle Details

The vehicle detail page shows:

- Vehicle information and metadata.
- Linked opportunities that reference this vehicle.

> **Note:** Only the vehicle creator or an admin can edit or delete a vehicle.
