# Compliance Tracking

The Compliance section tracks business certifications required for government contracting set-aside programs.

## Certification Types

| Type | Description |
|------|-------------|
| **Small Business** | SBA small business designation |
| **8(a)** | SBA 8(a) Business Development Program |
| **HUBZone** | Historically Underutilized Business Zone |
| **WOSB** | Women-Owned Small Business |
| **SDVOSB** | Service-Disabled Veteran-Owned Small Business |
| **EDWOSB** | Economically Disadvantaged Women-Owned Small Business |

## Certification Statuses

| Status | Description |
|--------|-------------|
| **Active** | Certification is current and valid |
| **Expiring Soon** | Within the expiration warning window |
| **Expired** | Certification has expired |
| **Pending** | Application submitted, awaiting approval |

## Adding a Certification

1. Navigate to **Compliance**.
2. Click **Add Certification**.
3. Fill in:
   - **Certification Type** -- Select from the types above.
   - **Issued By** -- Issuing authority.
   - **Issue Date** -- When the certification was granted.
   - **Expiration Date** -- When it expires.
   - **Status** -- Current status.
   - **Notes** -- Additional details.
4. Click **Save**.

## Expiration Alerts

The system monitors certification expiration dates and shows warnings when certifications are approaching expiry (configurable window, default 90 days). Expiring certifications are highlighted on the Compliance list page.

> **Note:** Only the certification creator or an admin can edit or delete compliance records.
