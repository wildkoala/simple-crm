# Teaming & Proposals

## Teaming

Teaming records track partner relationships for each opportunity.

### Teaming Roles

| Role | Description |
|------|-------------|
| **Prime** | Leading the effort as prime contractor |
| **Subcontractor** | Supporting under a prime contractor |
| **JV Partner** | Joint venture partner |

### Teaming Statuses

| Status | Description |
|--------|-------------|
| **Potential** | Exploring partnership |
| **NDA Signed** | Non-disclosure agreement executed |
| **Teaming Agreed** | Teaming agreement in place |
| **Active** | Actively working together |
| **Inactive** | Partnership no longer active |

### Managing Teaming Records

From an opportunity's detail page:

1. Navigate to the **Teaming** tab.
2. Click **Add Partner**.
3. Select a partner account, their role, and current status.
4. Add notes about the arrangement.
5. Click **Save**.

Partners must be registered as accounts first. If the partner doesn't exist, create them under [Accounts](./accounts.md) with the appropriate type (Partner, Prime Contractor, or Subcontractor).

## Proposals

Each opportunity can have one associated proposal that tracks the proposal development process.

### Proposal Statuses

| Status | Description |
|--------|-------------|
| **Not Started** | Proposal work has not begun |
| **In Progress** | Actively writing the proposal |
| **Review** | Under internal review (Pink/Red team) |
| **Final** | Finalized and ready for submission |
| **Submitted** | Proposal has been submitted |

### Managing a Proposal

Proposals are automatically created when an opportunity moves to the "Proposal" stage. You can also create them manually:

1. Navigate to the opportunity's detail page.
2. Go to the **Proposal** tab.
3. Set the proposal fields:
   - **Proposal Manager** -- User responsible for the proposal.
   - **Submission Type** -- Full, Partial, or Draft.
   - **Submission Deadline** -- When the proposal is due.
   - **Status** -- Current proposal status.
   - **Notes** -- Proposal-specific notes.
4. Click **Save**.
