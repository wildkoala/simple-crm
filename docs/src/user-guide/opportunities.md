# Opportunities & Pipeline

Opportunities are the core of the capture management workflow. Each opportunity tracks a potential contract through an 8-stage pipeline.

## Pipeline Stages

| Stage | Description |
|-------|-------------|
| **Identified** | Opportunity discovered (SAM.gov, agency forecast, tip) |
| **Qualified** | Evaluated and determined to be worth pursuing |
| **Capture** | Active capture activities (customer meetings, research) |
| **Teaming** | Building teaming arrangements with partners |
| **Proposal** | Writing and assembling the proposal |
| **Submitted** | Proposal submitted, awaiting award decision |
| **Awarded** | Contract won |
| **Lost** | Opportunity lost or decided not to bid |

## Creating an Opportunity

1. Navigate to **Opportunities** and click **Add Opportunity**.
2. Fill in the fields:
   - **Title** -- Descriptive name for the opportunity.
   - **Government Contract** -- Toggle for government vs. commercial.
   - **Agency** -- Issuing agency (for government contracts).
   - **Account** -- Link to an account.
   - **NAICS Code** -- North American Industry Classification System code.
   - **Set-Aside Type** -- Small Business, 8(a), HUBZone, WOSB, SDVOSB, Full & Open, or None.
   - **Estimated Value** -- Dollar value of the contract.
   - **Stage** -- Current pipeline stage.
   - **Capture Manager** -- Assigned user managing this pursuit.
   - **Win Probability** -- 0-100% likelihood of winning.
   - **Key Dates** -- Solicitation number, deadlines, expected release, proposal due, award estimate.
   - **Contract Vehicles** -- Link to applicable vehicles.
3. Click **Save**.

## Pipeline View

The **Pipeline** page displays opportunities as cards on a Kanban board organized by stage. Drag and drop cards between columns to update their stage.

## List View

The **Opportunities** page provides a filterable list with:

- Stage filter
- Agency filter
- NAICS code filter
- Set-aside type filter
- Source filter
- Value range filter
- Text search

## Opportunity Detail Page

The detail page is a full capture workspace with tabs for:

### Capture Notes

Structured sections for organizing capture intelligence:

- **Customer Intel** -- What you know about the customer's needs and preferences.
- **Incumbent** -- Information about the current contract holder.
- **Competitors** -- Known competitors and their strengths/weaknesses.
- **Partners** -- Potential and confirmed teaming partners.
- **Risks** -- Risks to winning or performing.
- **Strategy** -- Your win strategy and differentiators.

### Timeline

A chronological log of events related to the opportunity (discovery, meetings, RFP release, stage changes, etc.). Add events manually or let the system auto-log stage transitions.

### Teaming

Manage teaming relationships. See [Teaming & Proposals](./teaming-proposals.md).

### Proposal

Track proposal status and details. See [Teaming & Proposals](./teaming-proposals.md).

### Attachments

Upload and manage files (RFPs, SOWs, proposal drafts, etc.) related to the opportunity.

## Deleting an Opportunity

Opportunities use soft delete -- they're marked as deleted but not permanently removed. Admins can restore soft-deleted opportunities.
