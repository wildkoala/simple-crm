# Contracts

The Contracts section provides legacy contract tracking with a simple status workflow.

> **Note:** For full capture management, use [Opportunities](./opportunities.md) instead. The Contracts section is maintained for backward compatibility and simpler tracking needs.

## Contract Statuses

| Status | Description |
|--------|-------------|
| **Prospective** | Under evaluation |
| **In Progress** | Actively being worked |
| **Submitted** | Proposal submitted |
| **Not a Good Fit** | Decided not to pursue |

## Creating a Contract

1. Navigate to **Contracts** (available from sidebar or via direct URL).
2. Click **Add Contract**.
3. Fill in the fields:
   - **Title** -- Contract name.
   - **Source** -- Where the contract was found.
   - **Deadline** -- Submission deadline.
   - **Status** -- Current status.
   - **Submission Link** -- URL to the submission portal.
   - **Assigned Contacts** -- Link contacts to this contract.
   - **Notes** -- Free-text notes.
4. Click **Save**.

## SAM.gov Import

Contracts can be imported from SAM.gov:

1. Search SAM.gov for opportunities by NAICS code.
2. Select opportunities to import.
3. The system creates contract records and optionally creates contacts from the points of contact listed.
