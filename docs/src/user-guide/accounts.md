# Accounts

Accounts represent organizations you interact with. Each account has a type that reflects its role in the GovCon ecosystem.

## Account Types

| Type | Description |
|------|-------------|
| **Government Agency** | Federal, state, or local agencies that issue contracts |
| **Prime Contractor** | Companies that hold prime contracts |
| **Subcontractor** | Companies that work under prime contractors |
| **Partner** | Strategic partners for teaming arrangements |
| **Vendor** | Suppliers and service providers |

## Creating an Account

1. Navigate to **Accounts**.
2. Click **Add Account**.
3. Fill in the required fields:
   - **Name** -- Organization name.
   - **Account Type** -- Select from the types above.
4. Optionally fill in:
   - **Parent Agency** -- For sub-agencies (e.g., "Department of Defense" for "DARPA").
   - **Office** -- Specific office or division.
   - **Location** -- Physical location.
   - **Website** -- Organization's website URL.
   - **Notes** -- Free-text notes.
5. Click **Save**.

## Viewing Account Details

Click on any account to see its detail page, which includes:

- Account information and metadata.
- **Contacts** associated with the account.
- **Teaming records** referencing this account as a partner.

## Editing & Deleting

- Click **Edit** on the account detail page to modify fields.
- Click **Delete** to remove the account. This action is audited.

> **Note:** Only the account creator or an admin can edit or delete an account.
